# pyimaskill 使用说明

## 安装

```bash
pip install pyimaskill
```

开发环境：

```bash
git clone https://github.com/AJayBBB/pyimaskill.git
cd pyimaskill
pip install -e ".[dev]"
```

## 认证

`pyimaskill` 使用 `client_id + api_key` 方式认证，通过 `ImaClient` 初始化时传入。

```python
from pyimaskill import ImaClient

client = ImaClient(client_id="your-client-id", api_key="your-api-key")
```

## 快速开始

### 笔记操作

```python
from pyimaskill import ImaClient

with ImaClient(client_id="your-client-id", api_key="your-api-key") as client:
    # 搜索笔记
    result = client.notes.search(query="Python")
    for note in result.docs:
        print(note.doc.basic_info.title)

    # 创建笔记
    doc_id = client.notes.import_doc(content="# 标题\n\n正文")

    # 读取笔记
    content = client.notes.get_content(doc_id)
    print(content)
```

### 知识库操作

```python
from pyimaskill import ImaClient

with ImaClient(client_id="your-client-id", api_key="your-api-key") as client:
    # 列出知识库
    kb_list = client.knowledge.search_knowledge_base(query="")
    for kb in kb_list.info_list:
        print(kb.name)

    # 导入网页
    client.knowledge.import_urls(
        knowledge_base_id="kb_xxx",
        urls=["https://example.com/article"],
    )

    # 上传文件
    result = client.knowledge.upload_file(
        file_path="report.pdf",
        knowledge_base_id="kb_xxx",
    )
```
